Noir Editorial

A dark-mode design language built for data-rich editorial dashboards. Film grain texture, sharp typography, and vivid accent colors on deep black surfaces.

00 ‚Äî Principles

Design Rules

The non-negotiable rules that keep everything feeling cohesive. Break one and you break the system.
üåë
Dark First, Always
The background is near-black (#0a0a0f), not grey, not navy. Surfaces layer upward in brightness: black ‚Üí surface ‚Üí elevated. Never use pure white backgrounds.
‚ú¶
Grain is Mandatory
The SVG noise texture is applied as a fixed overlay at 4% opacity on the entire page (via body::before, pointer-events: none, z-index: 1000). This adds warmth and texture to every screen. Never remove it.
üî§
Three Fonts, Strict Roles
Playfair Display for headlines and big numbers only. DM Sans for all body copy, UI labels, and buttons. JetBrains Mono for data, metadata, tags, and code. Never swap roles.
üéØ
One Dominant Accent Per Context
Each section, card, or data category gets one accent color. Use its full range (dark ‚Üí base ‚Üí light ‚Üí glow ‚Üí subtle). Don't mix multiple bright accents in the same component.
üìè
Minimum Readable Size: 0.85rem / 13.6px
Body text must never go below 0.85rem. Metadata labels (mono font) may go as low as 0.7rem but only for non-essential supplementary info. Big numbers and headlines should be generous ‚Äî clamp() for responsiveness.
üí´
Subtle Motion, Not Spectacle
Use scroll-triggered fade-ups (0.8s cubic-bezier(0.16, 1, 0.3, 1)) and hover lifts (translateY(-2px)). Ambient glow pulses for atmosphere. Never bounce, never shake, never autoplay carousels.
‚óªÔ∏è
Borders Are Whispers
Borders use rgba white at 6‚Äì16% opacity. Never use solid grey or colored borders on containers. Colored top-borders (2px) are used only to tag category on cards.

03 ‚Äî Texture & Atmosphere

Surface Treatments

The grain overlay is the signature. Combine it with ambient color glows, subtle borders, and layered transparencies to create depth without heavy shadows.
Film Grain Overlay
body::before { background: var(--grain); position: fixed; z-index: 1000; pointer-events: none; }
Ambient Glow
radial-gradient(ellipse, color-glow 0%, transparent 70%) + filter: blur(60-80px) + pulse animation
Surface Fade
linear-gradient(180deg, --bg-surface 0%, --bg-black 100%) for section transitions
Category Card
border-top: 2px solid accent; Corner wash at 5% opacity for subtle color reinforcement
‚úì Do
Layer grain over glows. Use glow blobs positioned off-screen (-10% to -20%) for atmospheric bleed. Animate glow opacity between 0.6‚Äì1.0 on a slow 6s ease-in-out cycle. Use rgba borders (never solid colors) for container edges.
‚úó Don't
Use box-shadow for depth (it looks artificial on dark backgrounds). Use gradient backgrounds on cards (keep cards solid surface colors). Use more than two ambient glows per viewport. Use grain opacity above 5% ‚Äî it becomes distracting.

05 ‚Äî Motion

Animation Patterns

Restrained, purposeful motion. Everything fades up on scroll. Cards lift on hover. Ambient elements pulse slowly. That's it ‚Äî nothing else needed.
‚Üë
Scroll Reveal
opacity: 0 ‚Üí 1, translateY(30px ‚Üí 0). Duration: 0.8s. Easing: cubic-bezier(0.16, 1, 0.3, 1). Trigger: IntersectionObserver at threshold 0.1. Use staggered delays (0.1s increments) for grouped elements.
üñ±
Hover Lift
Cards: translateY(-2px) + background shift to surface-hover. Duration: 0.3s ease. Subtle ‚Äî don't overdo scale or shadow. No hover effects on mobile (use @media hover:hover).
üí´
Ambient Pulse
Glow blobs: opacity 0.6 ‚Üí 1.0, scale 1 ‚Üí 1.1. Duration: 6s ease-in-out infinite. Stagger second blob with a 3s delay. Only use in hero sections or major feature areas.
üìä
Data Bars
Width transition: 1.5s cubic-bezier(0.16, 1, 0.3, 1). Triggered when bar enters viewport. Start from width: 0, animate to final percentage. Creates satisfying "fill" effect.
/* Core easing curve ‚Äî use everywhere */ --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1); /* Scroll reveal class */ .reveal { opacity: 0; transform: translateY(30px); transition: all 0.8s var(--ease-out-expo); } .reveal.visible { opacity: 1; transform: translateY(0); } /* Stagger children */ .stagger-1 { transition-delay: 0.1s; } .stagger-2 { transition-delay: 0.2s; } .stagger-3 { transition-delay: 0.3s; } /* Ambient glow */ @keyframes pulse { 0%, 100% { opacity: 0.6; transform: scale(1); } 50% { opacity: 1; transform: scale(1.1); } } /* IntersectionObserver JS */ const observer = new IntersectionObserver((entries) => { entries.forEach(e => e.isIntersecting && e.target.classList.add('visible')); }, { threshold: 0.1, rootMargin: '0px 0px -50px 0px' }); document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
